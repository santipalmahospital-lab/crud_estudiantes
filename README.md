# Gestión de Estudiantes con Tkinter y SQLite

Este proyecto implementa un sistema CRUD (Crear, Leer, Actualizar, Eliminar) para gestionar estudiantes usando **Python**, **Tkinter** y **SQLite**.

## 📌 Funcionalidades
- Registrar estudiantes (nombre, email, edad, curso).
- Ver lista de estudiantes con sus datos.
- Eliminar estudiantes por ID.
- Actualizar estudiantes por ID.

## ⚙️ Requisitos
- Python 3.x
- Tkinter (incluido en Python)
- SQLite3 (incluido en Python)
